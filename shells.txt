Shell project are the last projects in my inner circle studies at Hive. We start my implementing a simple command interpreter (minishell) with input reading and some build-in functions as well as ability to execute programs
After that we continue to better our code with two group projects, 21sh and 42sh, resulting in a POSIX compliant shell in the end. I am now working on 21sh with a partner, our project can be found in my <a href="https://github.com/hlaineka/21sh">Github page</a>
In my minishell I consentrating on implementing some extra features when reading keyboard input, including left and right arrows for text editing and arrow up for history. I also implemented the asked build-ins with a lot of details and flags so that they represent the original commands as well as possible.
You can find my minishell implementation also in my <a href="https://github.com/hlaineka/minishell">Github page</a>
